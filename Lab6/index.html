¿Qué diferencias y semejanzas hay entre Java y JavaScript?

  • Java crea aplicaciones que tra ser compiladas se pueden ejecutar en navegadores a traves de una maquina virtual, mientras que JavaScript se ejecuta nativamente en los navegadores.
  • Java es un lenguaje fuertemente tipado y JavaScript es debilmente tipado.
  • Java es un lenguaje de programación orientado a objetos puramente, mientras que JavaScript es basado en prototipos.
